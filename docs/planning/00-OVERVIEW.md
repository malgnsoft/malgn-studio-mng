# 서비스 개요 · 비전 — 맑은스튜디오 (LMS AI Studio)

> 본 문서는 기획 패키지(01~12)의 **종합 요약**이다. 각 영역의 정본은 해당 번호 문서를 따른다.
> 작성일: 2026-06-17

---

## 1. 한 문장 정의

> **맑은스튜디오는 내 자료를 근거로(출처 표기) 영상·오디오·슬라이드·퀴즈까지 한 번에 만들고, 사람이 검수해, 자사 LMS와 표준(SCORM/xAPI)으로 바로 발행하는 — 국내 유일의 신뢰형 AI 학습 콘텐츠 빌더다.**

각종 자료(문서·텍스트·PPT·홈페이지 링크·영상)를 입력받아 **NotebookLM 수준**으로
학습 콘텐츠(학습 영상·오디오·슬라이드·커리큘럼·콘티·퀴즈·문제은행·요약)를 **AI로 제작**하는 통합 빌더.

---

## 2. 왜 지금, 왜 우리 (TL;DR)

- **생성은 이미 평준화됐다.** AI 초안 작성은 거의 모든 학습 플랫폼이 제공 → 생성 기능만으로 차별화 불가(01-COMPETITION).
- **국내엔 "통합 빌더"가 빈자리.** 조각(영상 생성·요약/자막·LXP 개인화)만 존재. 단 휴넷이 같은 방향 진입 중 → **속도가 경쟁력**.
- **우리의 해자 = 4가지의 결합**(단독은 글로벌 강자가 보유, 결합+자사 LMS 발행이 빈자리):
  1. **자사 LMS 네이티브 발행 + 표준 패키징**(SCORM/xAPI) — 종속 없이 양면 공략
  2. **스킬 기반 개인화** — 스킬 택소노미(입장권) + **생성형 변주**(차별점)
  3. **소스 그라운딩 · 출처 표기** — NotebookLM급 신뢰성, 환각 차단(국내 빈 메시지)
  4. **사람 검수(HITL)** — "출처 있는 초안 → 사람 승인"으로 신뢰 완성

---

## 3. 차별화 한눈에

| 축 | 시장 현황 | 맑은스튜디오 |
|---|---|---|
| 콘텐츠 생성 | 상향 평준화(기본) | 통합 멀티모달 + 교수설계 정렬 |
| 영상·오디오 | 도구 강함 | **외부 API 연동(Buy)** — 직접 개발 안 함 |
| LMS 발행 | 자체 플랫폼 종속 | **자사 LMS 네이티브 + 표준(SCORM/xAPI)** |
| 개인화 | 큐레이션 추천 | **큐레이션 + 생성형 변주** |
| 신뢰성 | 출처 표기 약함(국내) | **NotebookLM급 그라운딩 + 출처** |
| 품질 | 자동 생성 | **사람 검수(HITL) 게이트** |

## 4. 핵심 가치 체인 (MVP)

```
단일 소스(문서·PPT) → RAG 인덱싱·출처 → 산출물 3종(요약·슬라이드·퀴즈)
   → HITL 승인 → 자사 LMS 네이티브 발행 + SCORM 1.2  (+ 스킬 태깅 씨앗)
```

> MVP는 위 한 줄 체인으로 **4대 해자 씨앗을 모두 심는다**(03-FEATURES §5). 영상·오디오·커리큘럼 등은 Phase 2.

---

## 5. 시스템 구성 (형제 레포)

| 레포 | 역할 |
|---|---|
| `malgn-studio` | 사용자 스튜디오 앱 — 소스 업로드·생성 워크스페이스·산출물 편집·출처·검수·발행 |
| `malgn-studio-admin` | 운영자 콘솔 — 회원·콘텐츠·권한·과금·사용량·스킬 택소노미 |
| `malgn-studio-api` | 백엔드 + **AI 생성 파이프라인**(인제스트·RAG·LLM 오케스트레이션·외부 미디어 API) |
| `malgn-studio-lms` | 학습 관리 — 코스·수강·진도·평가(xAPI/LRS) |

> 본 레포 `malgn-studio-mng`는 위 4개 형제 레포의 개발을 **조망·관리하는 프로젝트 관리 허브**(대시보드·WBS·이슈·문서·이력).

---

## 6. 타깃 (가정)

기업교육(HRD/L&D, 1순위) → 공공·평생교육(2) → 대학/평생교육원(3). 코딩·IT 실습(엘리스 특화)은 초기 비대상.

---

## 7. 주요 리스크 (요약)

- 휴넷 선(先) 고도화 → **속도 선점**으로 대응(MVP를 약점 3지점에 집중).
- 외부 미디어 API 종속 → **어댑터 추상화 + 멀티벤더/제휴**.
- 그라운딩 품질 미달 → **HITL 검수 안전망 + 출처 강제**.
- 스킬 택소노미 부재(가정) → MVP는 태깅 씨앗만, 본격 개인화는 P2~P3.
