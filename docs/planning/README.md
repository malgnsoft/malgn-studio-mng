# 맑은스튜디오 서비스 기획 (정본)

맑은스튜디오 = 각종 자료(문서·텍스트·PPT·웹링크·영상)를 입력받아 **NotebookLM 수준**으로
학습 콘텐츠(영상·오디오·슬라이드·커리큘럼·콘티·퀴즈·문제은행·요약)를 **AI로 제작**하는 통합 빌더(LMS AI Studio).

> 포지션: 생성은 기본, **해자는 ① LMS 네이티브 발행+표준(SCORM/xAPI) ② 스킬 기반 개인화 ③ 소스 그라운딩·출처표기(환각 차단) ④ 사람 검수(HITL)** 의 결합.
