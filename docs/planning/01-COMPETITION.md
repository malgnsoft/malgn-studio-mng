# 경쟁 환경 분석 — LMS/LXP AI 콘텐츠 빌더

> 목적: AI 학습 콘텐츠 빌더(맑은스튜디오 / LMS AI Studio) 기획을 위한 경쟁 환경 정리
> 범위: 글로벌 + 국내
> 기준 시점: 2026-06 조사 / 출처는 문서 하단 참고
> 정본: 이 문서는 기획 입력(정본)이며, 전략·기능·로드맵 문서가 이를 근거로 참조한다.

---

## 0. 핵심 요약 (TL;DR)

- **AI 콘텐츠 "생성" 자체는 이미 상향 평준화된 기본 기능이다.** 2026년 기준 거의 모든 주요 학습 플랫폼이 AI 초안 작성을 제공하므로, 생성 기능만으로는 차별화가 안 된다.
- 따라서 우리의 해자(moat)는 다음 네 가지의 결합에 있어야 한다:
  1. **자사 LMS 네이티브 발행 + 표준 패키징**(SCORM/xAPI)
  2. **스킬 기반 개인화**(스킬 택소노미 + 적응형 + 생성형 변주)
  3. **소스 그라운딩 / 출처 표기**(NotebookLM급 신뢰성, 환각 차단)
  4. **사람 검수(HITL) 기반 품질 보증**
- **국내엔 "통합 빌더"가 아직 빈자리.** 조각(영상 생성, 요약/자막, LXP 개인화)은 존재하나 전 과정을 하나로 꿴 제품은 부재. 단 휴넷이 같은 방향으로 진입 중 → **속도가 곧 경쟁력**.

## 1. 글로벌 경쟁사

### 1.1. AI 영상 특화 (Video-first)
영상은 직접 개발보다 API 연동이 현실적임을 보여주는 그룹.

- **Synthesia**: AI 영상 강의의 사실상 표준. 140+ AI 아바타, 120+ 언어. 개인 요금 월 $30 안팎부터. 정교한 출력·기업 도입 강점.
- **HeyGen**: 약 40개 언어. 빠르고 저렴한 숏폼에 강점, 장편 교육 영상에는 덜 적합.
- **Colossyan**: 장면·다중 발표자·언어 전환 등 학습 중심 워크플로 강점(교육 영상 특화).

> 시사점: 아바타/영상 합성은 외부 API(빌려 쓰기)가 합리적. 직접 개발 대상 아님.

### 1.2. AI 코스 빌더 (올인원) — 가장 직접적 경쟁군
- **Coursebox**: AI 코스 생성 + 화이트라벨 LMS/LXP + 영상 + 퀴즈를 한 플랫폼에 통합. PDF/슬라이드 업로드 → 코스 자동 생성. 무료 플랜 + 크레딧 기반(텍스트/이미지/영상 분량) 요금제. 한국어 페이지로 국내 진입 중.
- **Articulate 360**: AI 코스 아웃라인 빌더 + 생성형 이미지·텍스트 추가. 전통 오서링 강자(Storyline/Rise).
- **360Learning**: 업로드 문서에서 코스 초안 자동 생성 + 퀴즈 제안. 협업·소셜 러닝 강점.
- **CYPHER Learning**: 학습목표에 정렬된 평가 자동 생성.

### 1.3. AI 네이티브 LXP / 엔터프라이즈 — 개인화 격전지
- **Sana**: LMS·LXP·오서링·가상교실을 통합한 AI 네이티브 플랫폼. 출처 표기된 실시간 답변(just-in-time). **2025년 Workday가 약 $1.1B에 인수** → 시장 신호 강함.
- **Docebo**: 'All-AI' 플랫폼 공개(동적 콘텐츠 생성, 가상 코칭, 협업 콘텐츠 설계). 콘텐츠 자동 태깅 + 스킬 태깅으로 스킬 갭 매핑. AI Creator / AI Video Presenter / Harmony(에이전트 마켓플레이스).
- **Cornerstone**: Skills Graph가 5만+ 스킬을 학습 콘텐츠에 매핑. 적응형 알고리즘으로 지식 격차 식별 → 개인화 경로 + 난이도 자동 조정.

> 시사점: **스킬 택소노미/스킬맵은 엔터프라이즈 LXP의 기본 입장권**이 됨. 우리도 필수.

### 1.4. 소스 그라운딩 벤치마크
- **NotebookLM**: 직접 경쟁자가 아닌 "신뢰 기반 생성"의 기준점. RAG 그라운딩 + 출처 표기 + 오디오 오버뷰. Sana 등도 이 UX(출처 표기)를 핵심으로 채택.

### 1.5. 글로벌 비교표
| 구분 | 대표 | 콘텐츠 생성 | 영상 | LMS/LXP 발행 | 스킬·개인화 | 그라운딩/출처 |
|---|---|---|---|---|---|---|
| 영상 특화 | Synthesia·HeyGen·Colossyan | 스크립트 | 아바타 강함 | 약함(외부) | 약함 | 약함 |
| 올인원 빌더 | Coursebox·CYPHER·Articulate | 강함 | 보통~강함 | 자체 LMS | 보통 | 보통 |
| AI 네이티브 LXP | Sana·Docebo·Cornerstone | 강함 | 보통 | 자체 통합 | **강함** | 강함(Sana) |
| **우리 목표** | LMS AI Studio | 강함 | 외부 API | **네이티브+표준** | 생성형 변주 | NotebookLM급 |

## 2. 국내 경쟁사

### 2.1. 기업교육 LMS + AI — 가장 직접적 경쟁 (특히 휴넷)
- **휴넷 (위협도: 높음)**
  - 드래그&드롭 '러닝 빌더' 기반 온·오프 통합 LMS **'러닝메이커'** 보유 → 방향성이 우리와 정면 충돌.
  - **APL(AI Powered Learning)** 표방: 역량 진단 → 교육 → 현업 적용 전 과정 AI, 초개인화 학습 + 실시간 평가·피드백.
  - **AI 기능 대폭 강화한 LMS를 연내 고도화 예고** → 시간 경쟁.
  - 영업 모멘텀: 지난해 AI 교육 의뢰 기업 4배↑, 관련 매출 2배↑.
- **휴먼웨어즈 (위협도: 중간)**
  - 생성형 AI LMS: 영상·음성 요약, 자동 자막, 지능형 문제 자동 생성, 학습 데이터 분석, AI 멘토링.
  - 성격: 기존 영상에 부가하는 **보조 기능 중심**. 슬라이드·콘티·오디오까지 통합 생성하는 풀 멀티모달 빌더는 아님.
- **멀티캠퍼스 (삼성계열)**
  - 기업교육 콘텐츠·플랫폼 강자. AI는 아직 "생성형 AI 활용법" 교육 과정 쪽에 무게.

### 2.2. LXP / 실습 플랫폼 — 엘리스
- **엘리스 (위협도: 중간)**
  - **'엘리스LXP'**: AI 기반 맞춤형 실습 교육 시스템. 국내외 교육 자료 CMS + 영상교실 등 운영 기능.
  - **AI헬피** 튜터: GPT·클로드 등 커리큘럼 맞춤 AI 에이전트 선택 가능.
  - 학습 구조: 10분 내외 영상 + 실습 문제 → 코드 자동 채점(마이크로러닝).
  - 한계: **코딩·IT 실습 도메인 특화** → 범용 콘텐츠 생성 빌더와는 결이 다름.

### 2.3. AI 영상/콘텐츠 생성 도구 — 브루(Vrew)
- **브루 Vrew (VoyagerX) (위협도: 낮음 / 제휴 후보)**
  - 텍스트로 영상 만들기: AI가 스크립트·이미지·음성 자동 생성, 교육 콘텐츠 최적화.
  - 200+ AI 목소리, 자동 자막, 키워드→영상, 긴 강의→요약 숏폼.
  - 한계: 순수 영상 생성 도구 → LMS 발행·평가·스킬 태깅·개인화 없음.
  - 포지션: 경쟁자라기보다 **영상 파트 연동/제휴 후보**.

### 2.4. 국내 비교표
| 플레이어 | 멀티모달 생성 | 소스 그라운딩/출처 | LMS·LXP 발행 | 스킬·개인화 | 도메인 | 위협도 |
|---|---|---|---|---|---|---|
| 휴넷 | 빌더 보유, AI 생성 고도화 중 | 약함 | **자체 LMS 강함** | APL·초개인화 추진 | 범용 기업교육 | **높음** |
| 휴먼웨어즈 | 요약·자막·문제(보조형) | 약함 | 자체 LMS | 맞춤 피드백 | 기업·공공 | 중간 |
| 엘리스 | 영상·실습(특화) | 보통 | **LXP 보유** | AI튜터·자동채점 | 코딩·IT 실습 | 중간 |

## 3. 포지셔닝 시사점 (통합)

1. **속도가 곧 경쟁력.** 휴넷이 LMS·콘텐츠·영업력을 갖춘 채 연내 AI LMS 고도화를 예고 → MVP는 휴넷이 약한 지점(그라운딩 신뢰성 + 멀티모달 통합 생성 + 표준 패키징)으로 빠르게 차별화.
2. **그라운딩·출처 표기는 국내에서 비어 있는 차별점.** 국내 AI LMS는 대부분 요약·자막·문제 생성에 머무름. "업로드 자료 기반 + 출처 매핑으로 환각 차단"을 전면에 내세운 곳이 거의 없음 → 교육 신뢰성 마케팅 메시지.
3. **영상은 국산 도구와 제휴가 합리적.** 브루 등 한국어 교육 영상 최적화 도구가 이미 강함 → 직접 개발보다 연동, 우리는 발행·개인화 가치에 집중.
4. **스킬 택소노미는 필수 입장권.** Docebo·Cornerstone이 자동 태깅·스킬맵을 기본 탑재 → 기존 LXP 스킬 프로파일/태깅 보유 여부 확인이 시급. 없으면 구축이 LXP 개인화의 전제.
5. **범용 도메인으로 빈 시장 공략.** 엘리스(코딩 특화)가 못 덮는 일반 기업교육·공공·평생교육에서 스킬 기반 개인화를 잘 풀면 선점 가능. 디지털 교육 시장은 2024~2029 연평균 31.4% 성장 전망.

## 4. 우리 차별화 전략 (요약)

| 축 | 시장 현황 | 우리의 포지션 |
|---|---|---|
| 콘텐츠 생성 | 상향 평준화(기본) | 통합 멀티모달 + 교수설계 정렬 |
| 영상 | 글로벌/국내 도구 강함 | 외부 API 연동(빌려 쓰기) |
| LMS/LXP 발행 | 자체 플랫폼 종속 | **자사 LMS 네이티브 + 표준(SCORM/xAPI)** |
| 개인화 | 큐레이션 추천 위주 | **큐레이션 + 생성형 변주** |
| 신뢰성 | 출처 표기 약함(국내) | **NotebookLM급 그라운딩 + 출처** |
| 품질 | 자동 생성 위주 | **사람 검수(HITL) 게이트** |
