# 작업 이력 (History)

맑은스튜디오 관리 허브의 날짜별 작업 이력 인덱스다. 각 항목은 `history.YYYYMMDD.md` 파일에 한 줄 요약·번호 섹션·산출물·다음 단계 순으로 기록한다. 날짜는 ISO(`YYYY-MM-DD`).
